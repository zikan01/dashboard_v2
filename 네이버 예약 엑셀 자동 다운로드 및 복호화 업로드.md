# 네이버 예약 엑셀 자동 다운로드 + 비밀번호 복호화 업로드 (요구사항 정리)

작성일: 2026-07-17 · 상태: 정리 완료, 개발 미착수

## 개요

네이버 스마트플레이스에서 예약 상세 엑셀을 자동으로 내려받아, 파일 비밀번호를 복호화한 뒤
대시보드에 자동 업로드하는 파이프라인. **로그인만 대표가 직접** 하고 나머지는 전 과정 자동.

## 실행 환경 (중요 전제)

- 최종 실행 환경은 **숲속할매들(대표) PC** — 클로드·크롬 제어 기능이 없는 독립 실행 환경
- 따라서 Playwright 등 **독립 실행형 스크립트**로 개발 (Claude/확장 의존 금지)
- 대표 PC: Windows 11, 브라우저는 네이버 웨일로 보임 (착수 시 확인)

## 역할 구분

### 대표가 직접 하는 일 (2가지)

1. 자동화용 바로가기 아이콘으로 브라우저 열기 (더블클릭 1회)
2. 네이버 로그인 — 세션 유지되면 생략, 만료 시에만 재로그인

### 자동으로 진행되는 부분 (스크립트)

1. 열려있는 브라우저에 연결 (디버그 모드로 실행된 브라우저에 접속)
2. 스마트플레이스 → 좌측 메뉴 "예약" 클릭 → 예약자관리 페이지 이동
   - `new.smartplace.naver.com` → `partner.booking.naver.com`
3. 조회 기간 설정 (기본: 이용일 기준 한 달)
4. 우측 상단 **"상세 내려받기"** 버튼 클릭 → 암호화 엑셀 다운로드
   - 주의: "내려받기"와 "상세 내려받기" 두 버튼 존재 — **상세 내려받기**가 대상
5. 다운로드 폴더 감시 → 새 파일 감지
6. **고정 비밀번호로 복호화** (비밀번호는 매번 동일 — 설정에 1회 등록, 자동 적용)
7. 대시보드 업로드 API 호출 → 예약 데이터 반영
8. 완료/실패 결과 알림

## 기술 방식

### 브라우저 제어: 요소 기반 (B안, 채택)

- 이미지 인식(PyAutoGUI) 대신, 디버그 모드로 열린 브라우저에 스크립트가 직접 연결해
  버튼을 **페이지 요소로** 찾아 클릭
- 장점: 해상도·창 위치·가림 여부 무관, 마우스 점유 없음, 버튼 디자인 변경에 강함
- 대표 사용법은 동일: "이 아이콘으로 브라우저 열고 로그인해두기"

### 비밀번호 복호화

- 네이버 다운로드 엑셀은 파일 암호화(암호 걸린 xlsx)
- 서버(또는 스크립트)에서 `officecrypto-tool`(Node) / `msoffcrypto-tool`(Python)로 복호화 → 일반 xlsx → 기존 파싱 로직 재사용
- 비밀번호는 **매번 동일** → 설정 화면에 1회 등록, 업로드 시 자동 적용

### 실행 시점 (택1, 착수 시 결정)

- A. 대표가 아이콘 클릭할 때마다 1회 실행
- B. 정해진 시간 자동 실행 (단, 브라우저가 켜져 있어야 함)

## 엑셀 컬럼 (상세 내려받기 기준, 화면 캡처로 확인)

상태 / 예약자 / 전화번호 / 예약번호 / 이용기간 / 객실 / 수량 / **옵션** / 요청사항 / 예약금 / 결제금액 / 신청일시 / 확정일시 / 취소일시

→ 옵션 컬럼 존재: `옵션 및 준비물 수기 입력.md`의 준비물 매칭 기능과 직접 연결

## 폴백 (자동 다운로드 불가 시)

다운로드만 대표가 수동으로 하고 지정 폴더에 저장 → 이후(감지→복호화→업로드)는 자동 처리하는 반자동 방식
