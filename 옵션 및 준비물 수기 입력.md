# 옵션 및 준비물 수기 입력 (요구사항 정리)

작성일: 2026-07-17 · 상태: 정리만 완료, 개발 미착수

## 개요

사이드바 "데이터" 그룹에 **옵션 업로드** 메뉴를 신설한다.
옵션별 준비물을 수기로 등록해두면, 업로드된 엑셀의 옵션 항목과 자동 매칭되어
예약 상세 화면과 데이터 내보내기에 준비물이 함께 표시·출력된다.

## 기능 정의

### 1. 옵션 업로드 (신규 메뉴)

- 위치: 사이드바 `데이터` 그룹 — 엑셀 업로드 / **옵션 업로드(신규)** / 텍스트 문의 / 데이터 내보내기 / 업로드 이력 (QA #2: 업로드 계열 인접 배치로 통일, 2026-07-17)
- 입력 방식: 텍스트 직접 입력(수기)
  - 옵션: `바베큐`
  - 준비물: `고기, 숯, 집게, 장갑` (쉼표 구분)
- 등록된 옵션-준비물 목록 조회 / 수정 / 삭제 가능해야 함

### 2. 엑셀 옵션 매칭

- 업로드된 엑셀 파일의 옵션 항목 값과 등록된 옵션명을 매치
- 매칭된 준비물을 해당 예약에 연결

### 3. 예약 상세 화면 표시

- 예약 상세에서 해당 예약의 옵션에 매칭된 준비물 확인 가능
- 예: 옵션 `바베큐, 계곡` → 각 옵션의 준비물 목록 표시

### 4. 데이터 내보내기 반영

- 내보내기 파일(엑셀)에 옵션별 매칭 준비물이 함께 출력

## 예시

| 옵션 | 준비물 |
|------|--------|
| 바베큐 | 고기, 숯, 집게, 장갑 |

## 추후 결정 필요 사항

- 미등록 옵션이 엑셀에 있을 때 처리 방식 (미매칭 표시 / 등록 유도 등)
- 옵션명 매칭 기준 (완전 일치 vs 부분 일치·공백/표기 차이 허용)
- 내보내기 시 준비물 출력 형식 (별도 열 / 옵션별 병기)
- 예약 상세 6탭 중 어느 탭에 표시할지 (서비스 탭 예상)
